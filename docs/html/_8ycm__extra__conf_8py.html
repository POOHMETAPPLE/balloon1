string PlatformioAutogen = ".pioenvs/"

Platformio automatically copies over the libs you use after your first run.

Be warned that you will not receive autocompletion on libraries until after your first platformio run.

string PlatformioCore = "~/.platformio/packages/framework-arduinosam/cores/arduino/"

This links to the Platformio Arduino Cores.

This provides the core libs, such as Arduino.h and HardwareSerial.h

string PlatformioLibs = "~/.platformio/packages/framework-arduinosam/libraries/"

This will link directly to the Platformio Libs for Arduino.

Be warned that this can polute your namespace (in #include) and slightly increase startup time (while crawling the lib dir for header files). This will however allow you to complete for header files you haven't included yet.
